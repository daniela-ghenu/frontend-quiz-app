# Frontend Mentor - Frontend quiz app solution

This is a solution to the [Frontend quiz app challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/frontend-quiz-app-BE7xkzXQnU). Frontend Mentor challenges help you improve your coding skills by building realistic projects.

A simple web quiz application where users can select a quiz subject, answer multiple-choice questions, receive instant feedback on their answers, view their final score, and replay to choose another subject.
